# Changelog

All notable changes to this project are documented here. The format is based on

## [0.1.0] — 2026-07-07
### Added
- Initial public release of the Kakunin platform under **AGPL-3.0**.
- X.509 certificate issuance via AWS KMS, real-time behavioral risk scoring,
  sub-60s auto-revocation, CRL, and MiCA / EU AI Act compliance reporting.
- Public verification endpoint, WORM audit log, and webhook delivery.
